# McHacks Sustainability Prototype

## Overview
Modular real-time multi-object video annotation and tracking system built using Python and OpenCV.

Developed during McHacks 2026 in a 4-person Agile hackathon team.

- Implemented CSRT-based multi-object tracking
- Integrated confidence scoring logic
- Built dynamic bounding box visualization layer
